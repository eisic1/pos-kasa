# Point of Sale
## Description
```
This POS (Point of Sale) application allows users to transact and manage sales operations.
The application is designed with Node.js backend and classic HTML, CSS, Bootstrap and JavaScript frontend.
This application also contains written tests for the frontend application.
```
## Functionalities
```
- Creating and managing accounts.
- Adding and deleting products from the basket.
- Calculation of the total amount of the order.
- Simple and intuitive user interface for sellers and users.
```
## Installation
```
FOR BACKEND:
npm install -> to install dependencies
npm start -> for run app

FOR FRONTEND:
path: /html/glavna.html
```
